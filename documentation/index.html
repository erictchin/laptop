1 Overview

This software is a collection of separate features:

  • Installation of selected software and font packages with Homebrew, Homebrew Cask and other package managers.

  • Disk management that allows for the use of an external drive as the main storage mechanism and the computer as a working station for the data.

  • Backup to Amazon S3 with Duplicity.

  • This project includes the private keys. This way it is possible to retrieve backups and reproduce the whole machine from scratch.

    Encryption with GnuPG.

  • Credentials management with Authinfo.

  • Configuration of command-line tools such as Git and Bash.

  • Email management with getmail for fetching, msmtp for sending, Dovecot for local delivery with Sieve filtering, Bogofilter for content-based spam filtering, mu for indexing and mu4e as the email client.

  • Emacs configuration based on Prelude.

  • Customized /etc/hosts based on Dan Pollock’s configurations.

  • Templates for projects and basic diagramming with PlantUML and Graphviz.

  • Solarized colorscheme everywhere.
