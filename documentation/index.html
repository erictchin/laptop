1 Overview

The laptop executable provides a collection of separate features:

  • Installation of selected software and font packages with Homebrew, Homebrew Cask and other package managers.

  • Installation of a compose key ⎄ for macOS.

  • Backup to an external storage drive and Amazon S3 with Rsync and Duplicity.

  • Installation of custom /etc/hosts based on Dan Pollock’s configurations.

  • Fix $PATH environment variable for GUI applications.

2 Installation

On a fresh macOS installation, run:

$ curl https://git.leafac.com/laptop/plain/installer/install.sh | bash

3 Usage
